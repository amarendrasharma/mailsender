# Install

<a id="Install">

1. `composer install`

2. `npm install`

3. `npm run watch`

4. `php artisan serve` / `valet open`

5. `php artisan queue:listen`

> Sometime the .lock files sucks a lot. If so just do one favour `sudo rm -rf filename`
